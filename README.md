# 📚 MyLibrary - Информационная система управления личной библиотекой

## Описание проекта

**MyLibrary** - это современное веб-приложение для управления личной коллекцией книг. Система позволяет пользователям легко добавлять новые книги, отслеживать прочитанные произведения, искать книги и просматривать статистику.

### Основные возможности:

- ✅ Добавление и удаление книг
- ✅ Редактирование информации о книгах
- ✅ Отметить книги как прочитанные/непрочитанные
- ✅ Поиск книг по названию и автору
- ✅ Фильтрация по жанру и статусу чтения
- ✅ Просмотр подробной информации о книге в модальном окне
- ✅ Статистика по библиотеке (общее количество, прогресс чтения, распределение по жанрам, топ авторов)
- ✅ Адаптивный дизайн (работает на мобильных устройствах)
- ✅ Сохранение данных в localStorage

## 📋 Модули JavaScript

### storage.js

Отвечает за работу с локальным хранилищем. Содержит:

- `getData()` - получить все книги
- `saveData(data)` - сохранить книги
- `addBook(book)` - добавить новую книгу
- `deleteBook(bookId)` - удалить книгу
- `updateBook(bookId, updates)` - обновить книгу
- `getBookById(bookId)` - получить книгу по ID

### utils.js

Вспомогательные функции:

- `formatDate(timestamp)` - форматирование даты
- `countByReadStatus(books, read)` - подсчет по статусу
- `countByGenre(books, genre)` - подсчет по жанру
- `getUniqueGenres(books)` - список уникальных жанров
- `getWeeklyStats(books)` - статистика за неделю
- `validateField(value, fieldName)` - валидация поля
- `showError(message)` - показать сообщение об ошибке
- `showSuccess(message)` - показать сообщение об успехе
- `confirmDelete(itemName)` - запрос подтверждения

### main.js

Логика главной страницы:

- Инициализация приложения
- Отрисовка списка книг
- Фильтрация и поиск
- Переключение статуса прочтения
- Удаление книг
- Модальное окно с подробной информацией
- Редактирование книг

### form.js

Логика страницы добавления:

- Заполнение выпадающего списка жанров
- Валидация формы
- Обработка отправки формы
- Сохранение новой книги

### stats.js

Логика страницы статистики:

- Показ основной статистики (всего, прочитано, не прочитано)
- Прогресс бар чтения
- Распределение по жанрам
- Статистика за неделю
- Топ авторов

## 🚀 Инструкция по использованию

### Главная страница (index.html)

1. Все книги отображаются в виде карточек
2. Используйте поле поиска для поиска по названию или автору
3. Фильтруйте по жанру и статусу прочтения
4. Нажмите на карточку, чтобы увидеть подробную информацию
5. В модальном окне вы можете отметить как прочитанное, редактировать или удалить

### Добавление книги (add.html)

1. Нажмите "Добавить книгу" на главной странице
2. Заполните обязательные поля (название, автор, жанр)
3. Опционально добавьте URL обложки
4. Нажмите "Сохранить"
5. Вы будете перенаправлены на главную страницу

### Статистика (stats.html)

1. Просмотрите общую статистику (всего книг, прочитано)
2. Видите прогресс чтения в виде прогресс-бара
3. Распределение книг по жанрам
4. Статистика за последнюю неделю
5. Топ авторов в вашей библиотеке

### О проекте (about.html)

- Информация об авторе
- Контактные данные
- Описание проекта
- Технологии используемые в разработке

## 💾 Сохранение данных

Все данные сохраняются в `localStorage` браузера под ключом `"books"`. Это означает, что:

- История не теряется при закрытии браузера
- Данные остаются даже если вы закроете вкладку
- Данные хранятся локально, не отправляются на сервер

### Структура одной книги:

```javascript
{
  id: 1234567890,              // Уникальный идентификатор
  title: "Война и мир",        // Название
  author: "Лев Толстой",       // Автор
  genre: "Классика",           // Жанр
  image: "https://...",        // URL обложки
  read: false,                 // Статус прочтения
  createdAt: 1234567890000     // Дата создания
}
```

Для редактирования книги:

1. Нажмите на карточку книги
2. В модальном окне нажмите "✏️ Редактировать"

## 📊 Статистика

На странице статистики вы можете увидеть:

- **Всего книг** - общее количество в библиотеке
- **Прочитано** - количество прочитанных книг
- **Не прочитано** - количество непрочитанных книг
- **Процент прочтения** - прогресс в процентах
- **По жанрам** - распределение книг по жанрам
- **На этой неделе** - статистика за последние 7 дней
- **Топ авторов** - рейтинг авторов по количеству книг
